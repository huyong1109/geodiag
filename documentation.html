3. Library usage

Several useful NCL functions or procedures are written for researchers who need to cast a NCL script by themself. For the time being, users can load the script as:

load "$GEODIAG_UTILS/geodiag_plot_utils.ncl"

3.1 Plot China map

The calling pattern is as following example:

res = True
set_china_map(res) ; ---------- # 1
... ; other plotting resources
plot = ... ; plot the figure
config = True
config@gdShowProvince = False
; ---------- # 3 draw_china_map(wks, plot, config) ; ---------- # 2
The #1 statement will set the range of lat-lon coordinates, and some other basic setup of a map. Some configurations can be set like #2 statement.
