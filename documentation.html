3. Library usage

Several useful NCL functions or procedures are written for researchers who need to cast a NCL script by themself. For the time being, users can load the script as:

load "$GEODIAG_UTILS/geodiag_plot_utils.ncl"

3.1 Plot China map

The calling pattern is as the following example:

res = True
setup_china_map(res) ; ---------- #1
... ; other plotting resources
plot = ... ; plot the figure
config = True
config@isShowProvince = False ; ---------- #2
config@isShowSouthChinaSea = True
config@isAddMask = True
config@isShowRivers = True
attach_china_map(wks, plot, config) ; ---------- #3
draw(plot) ; DO NOT FORGET THIS STEP!!! ; ---------- #4
The #1 statement will set the range of lat-lon coordinates, and some other basic setup of a map. Some configurations can be set like #2 statement. The #3 statement will attach the China map onto the base plot, which can be just a map or any other plot like contours. Finally, do not forget to draw the base plot!
