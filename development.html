1. Diagnostics package development

Currently a MJO diagnostics package is under development. Although there exists a version, this package will be easier to use (No C shell scripts!) with just one command to run all or parts of the three levels. So it will definitely alleviate the burden of model development and evaluation. This package will follow the standard diagnostics procedures of US-CLIVAR MJO working group.
