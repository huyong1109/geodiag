Download package from here or clone the git repository as

$ git clone https://github.com/lasg-model/geodiag.git

It is recommended to use git way to get GEODIAG! By this way, you can use the update subcommand to keep GEODIAG up to date easily.

Install GEODIAG is easy as breeze by just sourcing the "setup.sh" in BASH configuration file. The following is just an example in .bashrc:

source <path_to_GEODIAG>/setup.sh

For Linux newbie, <path_to_GEODIAG> is the directory where you put GEODIAG, e.g. /opt/geodiag, so change it please!

This will put the <path_to_GEODIAG> into PATH, set the environment variables (e.g., GEODIAG_ROOT), and setup auto-completion. Next some tool components (e.g. EEMD) written in C or other languages should be built by running:

$ geodiag warmup

Previously, this build stage is in setup.sh, but some user reported the echo command caused problem with scp, so I moved it into warmup subcommand.
